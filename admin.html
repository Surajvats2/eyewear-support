🖼️ Product Image Kaise Add Karein?

2 aasan tarike hain:

Option 1 — Free Image URL (Recommended):
1. Google Images pe apna product dhundho
2. Image pe right click → "Copy image address"
3. Ya pexels.com pe free stock photos milte hain
4. Woh URL neeche "Image URL" field mein paste karo

Option 2 — Google Drive:
1. Google Drive pe photo upload karo
2. File pe right click → "Share" → "Anyone with link"
3. Link se file ID copy karo: drive.google.com/file/d/FILE_ID/view
4. URL banao: https://drive.google.com/uc?id=FILE_ID
5. Yeh URL paste karo
